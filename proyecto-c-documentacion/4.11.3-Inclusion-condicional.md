# 4.11.3. Inclusión condicional

La **inclusión condicional** permite compilar partes del código solo si se cumplen ciertas condiciones, usando directivas como `#ifdef`, `#ifndef`, `#endif`.

---

## Sintaxis

```c
#ifdef DEBUG
    printf("Modo depuración\n");
#endif
```

- `#ifdef` incluye el bloque si la macro está definida.
- `#ifndef` incluye el bloque si la macro NO está definida.

---

## Ejemplo

```c
#define DEBUG

#ifdef DEBUG
    printf("Depuración activada\n");
#endif
```

---

## Comparación con JavaScript

- En JS, no existe compilación condicional nativa, pero se puede simular con herramientas de build.

---

## Resumen

- Usa inclusión condicional para depuración, portabilidad y modularidad.
